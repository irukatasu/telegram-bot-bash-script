# Telegram Bot Bash Script
#### Create commnand:

`chmod +x telegram-send.sh`

`sudo mv telegram-send.sh /usr/bin/telegram-send`

`sudo chown root:root /usr/bin/telegram-send`


##### Push Message:
`telegram-send "Test message" "GROUP_ID" "BOT_TOKEN"`
